# CAPTCHA-Free Passive Bot Detection

## Overview

This project is a machine learning-based passive bot detection system that identifies bots without using CAPTCHA. It analyzes user behavior and browser environmental data to distinguish between human users and automated scripts while maintaining a seamless user experience.

## Features

- CAPTCHA-free verification
- Behavioral analysis
- Browser fingerprinting
- Machine learning classification

## Technologies Used

- Python
- Flask
- JavaScript
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
